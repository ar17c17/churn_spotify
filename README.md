# Análisis de Churn en Spotify

Este repositorio contiene el desarrollo de una práctica de ciencia de datos enfocada en predecir la tasa de cancelación (*churn*) de los usuarios de Spotify.

## Descripción del Proyecto
El objetivo principal es identificar patrones de comportamiento que indican si un usuario dejará de utilizar la plataforma o cancelará su suscripción de pago. Se ha realizado un proceso de limpieza de datos, análisis exploratorio y preparación para el modelado predictivo.
